🐾 Paws API

Base URL: https://paws-api.jdog.dev

GET /pets

Returns all pets.

GET /pets

GET /pets/:id

Returns a single pet by ID.

GET /pets/1

POST /pets

Adds a new pet. Required fields: name, breed, description, age, fee.

POST /pets
Content-Type: application/json

{
  "name": "Luna",
  "breed": "Siamese Cat",
  "description": "Vocal and affectionate.",
  "age": 2,
  "fee": 160,
  "available": true,
  "goodWithKids": true,
  "image": "cat.png"
}
