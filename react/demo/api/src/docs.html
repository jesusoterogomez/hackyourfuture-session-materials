🐾 Paws API

Base URL: https://paws-api.jdog.dev

Pets

GET /pets

Returns all pets.

GET /pets

GET /pets/:id

Returns a single pet by ID.

GET /pets/1

POST /pets

Adds a new pet. Required fields: name, breed, description, age, fee.

POST /pets
Content-Type: application/json

{
  "name": "Luna",
  "breed": "Siamese Cat",
  "description": "Vocal and affectionate.",
  "age": 2,
  "fee": 160,
  "available": true,
  "goodWithKids": true,
  "image": "cat.png"
}

Pet object

FieldTypeDescription
idnumberAuto-assigned unique ID
namestringPet's name
breedstringBreed or species
descriptionstringShort description
agenumberAge in years
feenumberAdoption fee in Danish Kroner
availablebooleanWhether the pet is available
goodWithKidsbooleanKid-friendly?
imagestring | nullImage filename

Adoption requests

GET /requests

Lists all adoption requests, newest first. Returns a JSON array (empty array if none).

GET /requests

POST /requests

Creates an adoption request. Required: name (adopter) and petName. The pet name must match a pet in the catalog (case-insensitive). Server validates field length and that the pet exists.

POST /requests
Content-Type: application/json

{
  "name": "Amina",
  "petName": "Stepan"
}

DELETE /requests/:id

Deletes an adoption request by numeric id. Returns status 204 with no body on success, or 404 if the id does not exist.

DELETE /requests/1

Adoption request object

FieldTypeDescription
