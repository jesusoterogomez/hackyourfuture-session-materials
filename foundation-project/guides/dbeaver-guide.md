# DBeaver Guide - Connecting to SQLite

## Create a new connection

1. In DBeaver: **Database** → **New Database Connection**
2. Select **SQLite** and click **Next**
3. Click **Browse** and navigate to your `.db` file
4. Click **Finish**

## Run SQL queries

1. Right-click your connection → **SQL Editor** → **New SQL Script**
2. Type your SQL query
3. Press **Ctrl+Enter** (Windows) or **Cmd+Enter** (Mac) to run

## View table data

1. Expand your connection in the left panel
2. Expand **Tables**
3. Double-click a table to see its data

## Refresh after changes

If you made changes to your database and don't see them:

1. Right-click your connection
2. Click **Refresh**

Or use the keyboard shortcut, F5 in Windows, Fn+F5 in Mac to refresh
